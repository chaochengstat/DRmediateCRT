# mediateCRT: An R Package for Semiparametric Causal Mediation Analysis in Cluster-randomized Trial.

This package performs the semiparametric estimators in Cheng and Li (2024,arXiv:2404.18256) for mediation analysis in cluster-randomized trials.

A vignette is available at <https://rpubs.com/chaocheng/mediateCRT>.
